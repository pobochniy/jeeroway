# Краткое описание проекта
Гусеничная платформа. Имеет манипулятор и камеру. 
Можно подключиться с любого компьютера через интернет
Геймификация, обычным людям можно порулить реальным роботом.
Самая главная активность - разбор и сортировка незаконных свалок мусора. 
Это экологический it-проект, можно получать субсидии и гранты. 
Для инвесторов положены налоговые вычеты вплоть до 90%, налоговые льготы.
